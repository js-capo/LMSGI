Alineación de imágenes

La etiqueta img también soporta el atributo align, su uso no es recomendado, y se recomienda el uso de la propiedad float de CSS. Sin embargo esta propiedad aún no es soportada por todos los navegadores.

Para mantener un documento acorde al estricto uso de elementos XHTML 1.0 utilizaremos esta propiedad CSS en lugar del atributo align
Propiedad float de CSS

style="float:right"
